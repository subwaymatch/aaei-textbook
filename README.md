# AAEI Data Analytics Module

- University of Illinois Urbana-Champaign

> The Academy for Accounting Excellence and Innovation (AAEI) is designed to strengthen your skills in Excel, communication, and accounting fundamentals. This self-paced course allows you to create a flexible learning plan that fits your schedule while preparing you for success in the MAS program and your professional career.

This repository contains the textbook for the Data Analytics Module of the AAEI program.

## Development

This repository is built using [Jupyter Book Version 2](https://github.com/jupyter-book/jupyter-book), and is hosted on GitHub Pages.

### Install Jupyter Book

```bash
pip install jupyter-book
```

### Start Jupyter Book

```bash
jupyter book start
```
